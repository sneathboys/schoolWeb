What is Hypixel Skyblock?


Hypixel Skyblock is a mini game on the hypixel network. It has many different things you can do suck as skills in which u need to level to unlock new aspects of the game. These include: Combat - which you obtain by killing things, Mining - In which you obtain by mining, Foraging - in which you obtain by mining trees, Enchanting - In which you obtain by doing daily experiments using the experimentation table and Fishing- In which you obtain by fishing. These are core to the game as there are requirments to do cirtian things in the game and they give lots of skyblock exp.



Skyblock XP:


Skyblock XP is obtainable by doing many different things in the game. Every 100 xp you level up giving more health poibnts and more strength allowing more survivabilty aswell as damage.
